# 🧠 Intelligent Breast Cancer Classification System

A full-stack Machine Learning project implementing:

- Dimensionality Reduction (Filter + Wrapper Methods)
- Classification Models (SVM, RF, DT, KNN, NB)
- Model Comparison Dashboard
- SHAP Explainability
- Flask Backend API
- Streamlit Interactive Frontend

## 🔬 Techniques Used

### Filter Methods
- Correlation
- PCA
- LDA
- SVD

### Wrapper Methods
- SFS
- SBS
- SFFS
- SFBS

### Models
- Support Vector Machine (Best Performing)
- Random Forest
- Decision Tree
- KNN
- Naive Bayes

## 📊 Features
- Model Comparison Dashboard
- SHAP Waterfall Explainability
- Real-Time Feature Sensitivity Sliders
- Dataset Explorer
- Dimensionality Reduction Impact Visualization
